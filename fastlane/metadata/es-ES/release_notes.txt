Hemos añadido algunos cambios al editor de bloques para hacer que la experiencia de publicación sea aún mejor.

- Se acabó el copiar y pegar el texto alt—cuando añades un elemento en la biblioteca de medios a un bloque de imagen, el texto alt viaja con él.
- Puedes insertar medios desde una URL en los bloques de vídeo.
- Si un bloque contiene contenido inesperado o no válido, puedes usar la opción de recuperación de bloques para mostrar correctamente el contenido—sin tener que volver a añadirlo desde cero.

Los comentarios de las enetradas son también un poco más amigables. Cuando tecleas el símbolo @ para mencionar a otro usuario en tu comentario, el autor de la entrada o del comentario sale el primero en la lista de usuarios sugeridos. Perfecto.

Para finalizar, hemos corregido un problema con el selector de medios del edtior por el que las consultas de búsqueda podrían atascarse incluso antes de cerrar y volver a abrir la ventana. También hemos arreglado un fallo que hacía que la aplicación se bloqueara al añadir varios elementos a una entrada de historia.
