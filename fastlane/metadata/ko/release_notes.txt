블록 편집기의 갤러리/이미지 블록 성능을 약간 개선했습니다. 사이트에서 이미지가 차지하는 저장 공간을 줄이고 미디어 업로드 중 기기 속도가 느려지지 않도록 하는 데 도움이 됩니다. 속도는 중요합니다.

사이트에서 비디오프레스를 사용하는 경우 사이트의 웹 버전에 저장된 비디오 블록이 이제는 모바일 앱에서도 적절히 표시됩니다.

이제는 블록 편집기의 블록 삽입기를 사용하여 블록 유형을 모음에서 더 잘 정리할 수 있습니다. 작은 것도 중요하므로 이미지 블록의 대체 텍스트 각주 간격도 조금 조정했습니다.

무엇보다도, 사이트 생성 프로세스의 성가신 버그를 해결했습니다. 사이트 디자인 옵션을 가로로 스크롤할 때 이제는 페이지의 위아래로 스크롤하는 경우에도 중단한 위치를 앱에서 기억합니다.
